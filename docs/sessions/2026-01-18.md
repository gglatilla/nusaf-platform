### 3. Pricing Engine (src/lib/pricing/)
Implemented complete pricing calculation:

**For Imported Products:**
1. Start with supplier price (EUR)
2. Apply dealer discount if gross pricing
3. Convert EUR → ZAR using exchange rate
4. Add freight (12% sea, 30% air)
5. Calculate OEM price using margin factor
6. Calculate list price (OEM ÷ 0.60)
7. Calculate tier prices (30%, 40%, 50% off list)

**For Manufactured Products:**
- Raw material cost + machine time + labor
- Same margin/list price calculation

**For Assembled Products:**
- Sum of component costs + assembly labor
- Same margin/list price calculation

## Decisions Made

| Decision | Why |
|----------|-----|
| Prisma v5 instead of v7 | v7 has breaking changes requiring config migration; v5 is stable |
| Tailwind CSS v3 instead of v4 | v4 requires different PostCSS setup not compatible with current Next.js |
| Manual UI components vs shadcn CLI | More control, no external dependency, simpler setup |
| Exclude prisma/ from tsconfig | Seed script shouldn't be type-checked during Next.js build |
| Array.from() for Set iteration | TypeScript strict mode compatibility |

## Approaches Rejected

| Approach | Why Rejected |
|----------|--------------|
| Prisma v7 | Required significant config changes, datasource URL moved to config file |
| Tailwind CSS v4 | PostCSS plugin moved to separate package, compatibility issues |
| create-next-app | Conflicted with existing project files |
